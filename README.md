# HackerNews API Test Suite

Automated API acceptance tests for HackerNews using **Playwright** and **TypeScript**.

## Prerequisites
- Node.js
- yarn

## Setup
1. Clone the repository:
   `git clone https://github.com/yourusername/hackernews-api-test.git`
2. Install dependencies:
   `yarn`

## Running Tests
- To run all tests: `yarn run test` or `yarn playwright test`
- To see the HTML report: `yarn run report` or `yarn playwright show-report`
